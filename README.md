# ESAI - EcoSmart AI: Intelligent Sustainable Energy Agent

> Transform invisible energy waste into intelligent, automated, and measurable energy savings through AI, IoT sensors, and sustainable automation.

## 🌍 Mission

ESAI is an advanced AI-powered virtual assistant designed to optimize energy consumption, reduce electrical waste, automate smart environments, and promote sustainable energy practices in homes, schools, offices, and smart infrastructures.

### 3. **Productive Matrix Transformation**

## 🚀 Getting Started

### Prerequisites
- Python 3.10+
- Node.js 16+
- Docker & Docker Compose
- PostgreSQL 13+
- Firebase account
- OpenAI API key

### Installation

1. Clone the repository:
```bash
git clone https://github.com/estefcamila22-blip/EcoSmart-AI.git
cd EcoSmart-AI
```

2. Set up environment variables:
```bash
cp .env.example .env
# Edit .env with your configurations
```

3. Install dependencies:
```bash
pip install -r backend/requirements.txt
npm install --prefix frontend/
```

4. Start services:
```bash
docker-compose up -d
```
